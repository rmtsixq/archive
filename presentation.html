Measures overlap between

Binary Cross Entropy

Pixel-wise classification
loss function

BCE = -[y×log(ŷ) + (1-y)×log(1-ŷ)]
Combined Loss = 0.5 × Dice Loss + 0.5 × BCE Loss

Model Output

Probability Map

Each pixel value: 0.0 - 1.0

  • 0.5 = Medium probability
  • 0.0 = Low probability

Binary Mask

Threshold at 0.5

  • Probability ≤ 0.5 → No landslide
